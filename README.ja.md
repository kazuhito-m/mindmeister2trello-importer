Mindmeister to Trello Importer
==========================

## What's this ?

マインドマップ作成Webサービス「[Mindmeister](https://www.mindmeister.com/)」からエクスポートした標準JSONファイルを、
タスクボードWebサービス「[Trello](https://trello.com/)」のカードとして取り込むコマンドラインツール。
